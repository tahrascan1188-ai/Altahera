# نظام إدارة الطاهرة الطبي (Altahera Management System)

نظام إلكتروني متكامل لإدارة الفروع ومراقبة الأجهزة الطبية، وتتبع حركاتها وإشعاراتها، مع وحدات ذكية لإدارة الفحوصات (التحاليل والأشعة)، وجدولة مواعيد الأطباء. يعتمد النظام على واجهة مستخدم حديثة وسريعة الاستجابة (Responsive UI) ويعمل بالارتباط مباشرة بقاعدة بيانات سحابية (Google Sheets API) لضمان المركزية والمرونة الكافية.

---

## 🚀 المميزات الرئيسية (Key Features)

- **لوحة تحكم تفاعلية (Dashboard)**: احصائيات سريعة للمدراء لتعقب حالة الفروع ومتابعة النشاطات الحالية بشكل رسومي وواضح.
- **إدارة صلاحيات المستخدمين (RBAC)**: نظام دقيق لإدارة الصلاحيات يسمح بالتحكم بمستويات الوصول (مدير النظام الكلي، مدير فرع معين، موظف استقبال، فريق الكول سنتر) وضمان سرية البيانات.
- **مراقبة وتحكم بالأجهزة الطبية (Devices Monitoring)**: توثيق دقيق لحركة كل جهاز طبي بين الفروع، وتغيير حالات الجهاز (متاح للعمل، صيانة، خارج الخدمة)، مع إجبار المستخدم على تسجيل سبب التحويل أو العطل.
- **سجل مركزي وإشعارات فورية (Audit & Notifications)**: نظام إشعارات ذكي يُنبه المدراء فور حدوث أي تدوير أو عطل يخص أجهزة الفروع، بالإضافة إلى لوحة مخصصة لحفظ "أرشيف حركات الأجهزة" مُوثقاً بالموظف والتاريخ والسبب.
- **إدارة الفحوصات الطبية (Tests Directory)**: نافذة مخصصة لإضافة وتسعير تحاليل المعامل والفحوصات الإشعاعية وتحديثها بشكل دائم ومزامنتها على جميع الأجهزة.
- **إدارة جدول الأطباء (Doctors Scheduler)**: ترتيب أسبوعي لمواعيد ಅطباء الفروع وتوضيح حالة التوافر (حاضر، غائب، إجازة) بناءً على الجداول الفعلية.

---

## 🛠️ التقنيات المستخدمة (Technologies)

- **الواجهة الأمامية والمنطق (Frontend)**: 
  - (HTML5, Modern Vanilla CSS3, JavaScript (ES6+
  - أسلوب التصميم الحديث (Glassmorphism) المتوافق مع الهواتف الذكية والأجهزة اللوحية (Fully Responsive).
- **الخادم وقاعدة البيانات (Backend & DB)**:
  - الاعتماد على **Google Apps Script** كأداة API وسيطة للربط.
  - الحفظ السحابي والمباشر عبر جداول **Google Sheets** للوصول الآمن والمجاني وتخزين البيانات السريع (Serverless Architecture).
